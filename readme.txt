== Description ==

The WordPress Posts Timeline Plugin uses js and css to create a horizontal timeline for a custom post/page/posts like timeline.verite.co. Its tested and 100% work - <strong>RIGHT NOW ITS UNDER UPGRADATION</strong>

[Donate Here:](http://gojaipurgo.com/donate)

== Installation ==

1. Unzip wp_timeline.zip to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [history_timeline] in page or template to display timeline.


== Frequently Asked Questions ==

= How do I change the date format? =

The date is pulled from your posts published date. If you need to backdate, you can change it there.
